# Project Overview: CodeLog

## Purpose
CodeLog is a platform for developers to share code snippets and posts, with social features like following, liking, and commenting.

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Library**: React 19
- **Backend/Auth**: Supabase
- **Styling**: Tailwind CSS 4, Radix UI
- **Language**: TypeScript
- **UI Components**: Custom components based on Radix UI, Lucide icons.
- **Code Highlighting**: Prism.js

## Codebase Structure
- `src/app`: Next.js pages and layouts.
- `src/components`: Reusable UI components.
- `src/services`: Business logic and database interactions (Service-Interface pattern).
- `src/actions`: Next.js Server Actions.
- `src/types`: TypeScript interfaces and types.
- `src/utils`: Helper functions.
- `src/providers`: React context providers.
- `src/styles`: CSS and global styles.
